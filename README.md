# Network Communications (SCSR1213) 🔌🌐

Welcome to my portfolio repository for **SCSR1213: Network Communications**. This space contains a curated collection of network packet captures, communication protocol simulations, mathematical subnet blueprints, and an enterprise network infrastructure capstone design project.

The core focus of this course is understanding the fundamentals of data transmission across the layers of the OSI and TCP/IP reference stacks, moving from basic local host loopbacks to designing scalable, secure, and resilient enterprise wide-area ecosystems.

## 📋 Course Portfolio Core Components

| Repository Name | Summarized Content | Key Skills Developed |

## 🛠️ Core Technological & Theoretical Toolkit

* **Network Analysis & Emulation:** Wireshark Packet Analyzer, Cisco Packet Tracer (Simulation & Realtime Environments).
* **Reference Architectural Models:** OSI Reference Stack, TCP/IP Protocol Layering (Application, Transport, Network, Link, Physical Layers).
* **Core Communication Protocols:** HTTP, DNS (Type A Records), FTP (Control Port 21), SMTP, TCP (Handshaking & Windowing Flags), UDP (Port 53 Lookups), ARP.
* **Routing & Subnetting Mechanics:** Variable Length Subnet Masking (VLSM), Classless Inter-Domain Routing (CIDR), Routing Information Protocol (RIP).
* **Wireless & Security Standards:** Service Set Identifier (SSID) Broadcast, DHCP Auto-Address Allocation Pools, WPA2-Personal Advanced Encryption.
